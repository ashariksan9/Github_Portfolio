HiKopi! - Personalized Coffee Shop System Recommendations

HiKopi! is a recommendation app that helps users find coffee shops based on their preferences. Using data from PergiKuliner.com, we applied NLP techniques to analyze reviews and match cafes with user-defined criteria. The system processes keywords to deliver personalized recommendations, enhancing user experience with precise and relevant suggestions.

  • Selenium
  • NLP (Word2Vec)
  • Web Scraping

Automating Data Processing and Visualization for Employee Well-being

This project automates the analysis of employee well-being by integrating PostgreSQL, Great Expectations, and Apache Airflow for data processing. Cleaned data is stored in Elasticsearch and visualized with Kibana, revealing insights into work conditions and employee productivity. The results support better workplace policies for a healthier workforce. Feel free to visit the full project in the github repo links below!

Plastic, Paper, and Garbage Waste Detection (Computer Vision)

This project utilizes Convolutional Neural Networks (CNN) to classify waste into plastic, paper, and general garbage, enabling efficient recycling and sustainable waste management. The model processes image data to detect and categorize waste automatically, reducing manual sorting efforts. By improving waste classification accuracy, this solution contributes to a cleaner environment and optimized recycling workflows.

  • TensorFlow/Keras
  • OpenCV
